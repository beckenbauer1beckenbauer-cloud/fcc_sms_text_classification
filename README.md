# Neural Network SMS Text Classifier

This project is part of the freeCodeCamp Machine Learning with Python Certification. It utilizes a Deep Learning Neural Network built with TensorFlow and Keras to classify SMS messages as either **ham** (normal) or **spam** (advertisement/scam).

## Project Requirements
* Convert categorical labels (`ham`/`spam`) into numerical digits (`0`/`1`).
* Tokenize and apply padding to the text sequences.
* Build an `Embedding` layer architecture to capture semantic word meanings.
* Create a `predict_message()` function to predict unseen texts.
